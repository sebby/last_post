LAST POST
=========

LastPost is a module for the Drupal CMS,  

with LastPost we get the last tweet, the last article from RSS, last post in the Facebook Wall, etc.





Création d'un outil permettant de parser un fichier mxml provenant d'un flux RSS
Utilisation des classes simpleXMLElement et xPath.

Tester avec le flux "A la une" http://www.lemonde.fr/rss/une.xml 
prise sur le site lemonde.fr
